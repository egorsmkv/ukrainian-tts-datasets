Listen to [DEMO](https://huggingface.co/spaces/theodotus/ukrainian-voices) (choose "lada" in the Voice field)

## Features

- Quality: high
- Duration: 10h37m
- Audio formats: OPUS/WAV
- Text format: JSONL (a `metadata.jsonl` file)
- Frequency: 48000 Hz
